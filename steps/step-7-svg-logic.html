SVG Structure and Icon Logic

How SVG icons are built in this project

Apps Icon

Uses a 24x24 viewBox with one path. Dot positions are encoded in the path data.

Voice Icon

Uses the Material-style coordinate system. Large viewBox keeps native icon proportions.

Image Icon

Same viewBox scale as voice icon to keep consistent visual weight in the right tool group.

Left Search Icon

Placed inside a circular hover target. SVG stays 20x20 while click area is 32x32.

Rule 1: keep the original path untouched to preserve brand shape fidelity.

Rule 2: adjust size with CSS width and height on the svg, not by editing path data.

Rule 3: keep interactive area larger than icon area for better usability.
